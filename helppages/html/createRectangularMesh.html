createRectangularMesh
  Create a mesh structure with a rectangular geometry
 
  pmcmesh = createRectangularMesh(xsize, ysize, step)
 
  INPUT
 
   xsize:       the radius of the rectangle [mm]
   ysize:       discretisation step size [mm]
   dh:          step size [mm]
 
  OUTPUT
 
   pmcmesh:        structure that contains the geometry (triangles,
                                                      boundary
                                                      lines etc.)
