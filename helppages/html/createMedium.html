createMedium
  Complete a 'pmcmedium' -structure for a given geometry
 
  pmcmedium_out = createMedium(pmcmesh, pmcmedium_in)
 
 
  INPUT
 
   pmcmesh:  	    mesh structure (see documentation/structure reference)
 
  OPTIONAL INPUT
 
   pmcmedium_in:	existing medium structure (see documentation/structure reference)
                where the array size does not match the number of elements in
                the mesh.
  OUTPUT
 
   pmcmedium_out:  pmcmedium structure for a given mesh. The array sizes match
                the number of elements in the mesh.
                If 'pmcmedium_in' was given as a input, the values for the arrays in
                'pmcmedium_out' are obtained by repeating 'pmcmedium_in'. 
                If no 'pmcmedium_in' was given, the function returns a structure with
                the fields formatted as follows:
 
                    pmcmedium_out(:).refractive_index = 1;               
                    pmcmedium_out(:).scattering_coefficient = 0; [1/mm]
                    pmcmedium_out(:).absorption_coefficient = 0; [1/mm]
                    mediun_out(:).scattering_anisotropy = 0;
