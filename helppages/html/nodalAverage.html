nodalAverage
  Computes volume/area averages of element-wise constant values val,  defined for elements H in grid r. H can be either line, triangle, or tetrahedral topology.
  This is convenient for plotting with trimesh
  ret = nodalAverage(H, r, val);
 
  INPUT
 
  H        H can be either line, triangle, or tetrahedral topology. 
  r        grid point coordinates
  val      element-wise constant
 
  OUTPUT
 
  ret      val averaged for each node
