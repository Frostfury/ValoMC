createBoundary
  This function is used to construct a complete 'pmcboundary' -structure for a given geometry
 
  pmcboundary_out = createBoundary(pmcmesh, pmcboundary_in)
 
  INPUT
 
   pmcmesh:        mesh structure (see documentation/structure reference)
 
  OPTIONAL INPUT
 
   pmcboundary_in:	existing pmcboundary structure (see documentation/structure reference)
                where the array size does not match the number of boundaries in
                the mesh.
  OUTPUT
 
   pmcboundary_out: pmcboundary structure for a given mesh. If 'pmcboundary_in' was
                 given as a input, the values for the arrays in
                 'pmcboundary_out' are obtained by repeating 'pmcboundary_in'. If no
                 'pmcboundary_in' was given, the function returns a structure with
                  only 'lightsource' field. This is the minimum that is needed to
                  run the PhotonMC2D. The array size matches the number of boundaries
                  in mesh.
